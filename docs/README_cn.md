# DBSight

[English](../README.md) | **中文**

DBSight 是一款现代化、高性能的数据库可视化工具，为需要清晰展示数据库结构与数据的研发、运维、数据库管理员量身打造。

基于 **Rust**、**GPUI** 与 **gpui-components**，DBSight 拥有接近原生的图形性能，同时具备高稳定性与高安全性，为专业用户提供流畅、高效的数据库操作体验。

## 🚀 核心特性

- 支持多种主流数据库（MySQL、PostgreSQL、SQLite 等）
- 高性能表格渲染（虚拟滚动）
- 清晰易读的数据库结构浏览器
- 多标签 SQL 编辑器，快速执行查询
- 现代化、GPU 加速的界面设计

## 🎯 面向人群

- 后端工程师
- DevOps / SRE
- 数据库管理员（DBA）
- 任何需要高效数据库可视化工具的人

## 🌟 优势

- Rust 提供卓越的性能与安全保障
- 基于 GPUI，拥有真正的原生级图形渲染速度
- 轻量、现代、美观的 UI
- 可轻松扩展至更多数据库类型
